# Git teaching figures

Two illustrative figures for the Git & GitHub session, generated by
`make_git_figures.py` (matplotlib, PNG at 200 dpi).

| File | Shows |
|------|-------|
| `git_three_areas.png` | The three areas — workspace, index, commit history — with the commands that move changes forward (`add`, `commit`, `commit -a`) and back (`restore`, `restore --staged`, `switch`), plus the read-only inspection commands (`status`, `diff`, `log`). |
| `git_local_remote.png` | How a local clone talks to a remote (`origin` on GitHub): `clone`, `push`, `fetch` vs `pull`, and the `origin/main` remote-tracking branch. |

## Regenerate

```powershell
python make_git_figures.py
```

Requires only `matplotlib` (`pip install matplotlib`). Edit the script to
change wording, colors, or layout — all coordinates are in inches on a
12.5-inch-wide canvas.
